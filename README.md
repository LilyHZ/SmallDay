## 高仿小日子 - Swift2.0
- 使用Swift2.0小日子
- 编译器为Xcode7.0正式版,由于Swift语法还不稳定,每个版本都会出现语法修改,为保证程序的正常运行,请在App Store将Xcode更新到最新的Xcode7.0版本运行工程
- 直接打开SmallDay.xcworkspace运行工程即可
